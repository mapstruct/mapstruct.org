Getting Help

Edit on GitHub

You are using MapStruct and ran into a problem?

First check out the reference guide. If that doesn’t help to answer your question you may join the MapStruct GitHub Discussions or hop by the MapStruct Gitter room. We also monitor the mapstruct tag on StackOverflow.

To report a bug or request a new feature use the MapStruct issue tracker. Note that bug reports should be accompanied by a test case demonstrating the issue.
